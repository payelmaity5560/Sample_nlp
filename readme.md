# üß† NLP Preprocessing App using Streamlit

This project is a **Streamlit-based web application** that demonstrates **basic NLP preprocessing techniques** such as tokenization, text cleaning, stemming, lemmatization, and Bag of Words (BoW).

It is useful for **students, beginners, and NLP learners** to understand how text preprocessing works in real time.

---

## üöÄ Features

The application supports the following NLP techniques:

1. **Tokenization**
   - Sentence Tokenization
   - Word Tokenization
   - Character Tokenization

2. **Text Cleaning**
   - Lowercasing text
   - Removing punctuation and numbers
   - Removing stopwords using spaCy

3. **Stemming**
   - Porter Stemmer
   - Lancaster Stemmer

4. **Lemmatization** *(can be extended using spaCy)*

5. **Bag of Words**
   - Text vectorization using `CountVectorizer`
